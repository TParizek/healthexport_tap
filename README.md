# Homebrew Tap for HealthExport CLI

This repository contains the Homebrew formula for installing
[HealthExport CLI](https://github.com/TParizek/healthexport_cli).

Install:

```bash
brew tap TParizek/healthexport_tap https://github.com/TParizek/healthexport_tap
brew install TParizek/healthexport_tap/he
```

The `Formula/he.rb` file is updated by GoReleaser from the source repository.
